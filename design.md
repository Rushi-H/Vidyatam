# Vidyatam Classes – Design Guidelines

## Design Goal

Create a modern, premium, and visually engaging education website for Vidyatam Classes.
The design should inspire trust among parents and motivate students.

The UI should feel **clean, modern, and slightly futuristic with subtle 3D elements**.

---

## Color Scheme

Primary Color
Royal Blue #1E3A8A

Secondary Color
White #FFFFFF

Accent Color
Soft Yellow / Gold #FACC15

Background
Soft gradient from white to light blue.

---

## Layout Principles

* Clean and spacious layout
* Large readable headings
* Balanced white space
* Centered hero section
* Responsive grid system
* Section spacing for clarity

---

## Typography

Heading Font Style
Bold modern sans-serif

Body Font Style
Clean readable sans-serif

Text hierarchy:

H1 – Large hero headings
H2 – Section titles
H3 – Card titles
Paragraph – Supporting content

---

## UI Components

### Cards

Use rounded cards for courses and features.

Card Style:

* Rounded corners
* Soft shadow
* Glassmorphism effect
* Hover animation

---

### Buttons

Primary Button
Royal blue background with white text.

Hover Effect
Light glow or slight scale animation.

Example buttons:

* Enroll Now
* Contact Us
* Get Admission Details

---

### Icons

Use simple education-related icons:

* Book
* Graduation cap
* Pencil
* Brain / knowledge icon

Icons should be minimal and modern.

---

## 3D Visual Elements

Include subtle 3D elements such as:

* Floating books
* Graduation cap animation
* Moving education icons
* Soft animated shapes in background

These elements should create a **premium modern feel**.

---

## Animation Guidelines

Use smooth animations only.

Allowed animations:

* Fade-in sections on scroll
* Floating 3D elements
* Hover card animation
* Smooth parallax scrolling

Avoid overly flashy animations.

---

## Hero Section Design

Large central heading:

VIDYATAM CLASSES

Subheading:

Quality Education • Strong Foundation • Bright Future

Include:

* Two CTA buttons
* 3D floating education elements
* Soft animated background

---

## Section Layout Order

1. Hero Section
2. About Vidyatam Classes
3. Courses Offered
4. Why Choose Us
5. Student Achievements
6. Testimonials
7. Contact Section

---

## Mobile Design

The website must be fully responsive.

Mobile adjustments:

* Stacked sections
* Large buttons
* Readable font sizes
* Optimized animations

---

Professional
Modern
Premium
Education-focused
Friendly and inspiring
