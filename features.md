# Vidyatam Classes – Website Features

## Core Objective

The website should help Vidyatam Classes:

* attract new students
* provide course information
* allow admission enquiries
* build credibility

---

## Key Features

### 1. Hero Section

Includes:

* Title
* Tagline
* Short description
* Enroll Now button
* Contact button

Hero visual should include 3D education objects.

---

### 2. About Section

Explain the teaching philosophy of Vidyatam Classes.

Content should highlight:

* concept clarity
* experienced teachers
* supportive learning environment
* exam preparation guidance

---

### 3. Courses Section

Show available classes:


Each course card should include:

* title
* short description
* icon
* hover animation

---

### 4. Why Choose Us

Highlight institute strengths:

Concept Clarity
Regular Tests
Personal Guidance
Supportive Learning Environment

Each point should appear in animated feature cards.

---

### 5. Student Achievements

Display success indicators:

* high board results
* academic improvement
* student success stories

Optional elements:

* progress statistics
* achievement highlights

---

### 6. Testimonials

Include quotes from students or parents describing:

* teaching quality
* concept understanding
* improvement in marks

---

### 7. Contact Section

Include:

Phone number
WhatsApp contact button
Google map location
Admission enquiry form

---
