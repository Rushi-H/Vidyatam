# Vidyatam Classes Website – Context File

## Project Overview

This project is a modern 3D animated website for **Vidyatam Classes**, a coaching institute focused on providing quality education and strong academic foundations for students from 8th to 12th standard.

The goal of this website is to create a **premium, modern, and visually engaging education website** that attracts students and parents while clearly presenting courses, teaching methodology, and admission information.

The website should feel **professional, trustworthy, and inspiring for students**.
